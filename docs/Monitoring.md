# Monitoring

Amazon CloudWatch is used to monitor the application.

## Metrics

- CPU Usage
- Memory Usage
- Disk Usage

## Logs

- System Logs
- Nginx Access Logs
- Nginx Error Logs

## Alarm

A CloudWatch Alarm monitors CPU utilization.

If CPU exceeds the configured threshold, Amazon SNS sends an email notification.

## Benefits

- Detect failures
- Monitor resource usage
- Receive alerts
- Analyze system performance
